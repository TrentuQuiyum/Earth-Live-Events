5. **Initialize**: Leave all checkboxes empty
6. Click **Create repository**

### **STEP 2: Open Command Prompt**
```bash

```

### **STEP 3: Initialize Git Locally**
```bash
git init
git add .
git commit -m "Initial commit: Add Earth Live Events Dashboard"
```

### **STEP 4: Add GitHub Remote**
```bash
git remote add origin https://github.com/YOUR-USERNAME/earth-live-events.git
```
*(Replace YOUR-USERNAME with your actual GitHub username)*

### **STEP 5: Rename Branch**
```bash
git branch -M main
```

### **STEP 6: Push to GitHub**
```bash
git push -u origin main
```

**When prompted for authentication:**
- Let it open a browser
- Login with your GitHub account
- Grant permission
- Done!

### **STEP 7: Verify on GitHub**
